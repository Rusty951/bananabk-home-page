# Banana Black Website Renewal PRD v1

## 1. Project Overview

### Project Name
Banana Black Website Renewal

### Project Type
Portfolio website redesign and rebuild

### Goal
Rebuild the current Banana Black website from scratch so it looks more premium, more intentional, and more brand-driven while using mostly the same photo assets.

This is not a partial edit of the current site.  
This is a full redesign based on a new brand and UX direction.

### Core Objective
The current website shows the work, but it does not elevate the work enough.

The new website should:
- make the same photos feel more premium
- create a stronger first impression
- clarify category structure
- improve inquiry conversion
- support hidden portfolio sharing for private works
- feel like a real brand portfolio, not a basic gallery template

---

## 2. Current Diagnosis

### Current Strengths
- Photo quality is already strong
- Product category is especially competitive
- Food category also has good styling and color
- The current site is simple and easy to understand
- Category separation already exists

### Current Problems
1. Home is too weak
   - the hero area does not fully use the strongest images
   - the first impression is not strong enough

2. The site feels like a simple image grid
   - it looks functional but not premium
   - it does not feel curated enough

3. CTA is too weak
   - inquiry flow is passive
   - users are not guided naturally toward contact

4. Category character is not reflected in layout
   - Product, Food, Portrait, and Space all have different visual strengths
   - current structure treats them too similarly

5. Brand identity is not fully expressed
   - the site does not yet feel like a distinct creative studio brand

---

## 3. Brand Direction

### Brand Interpretation
Banana Black should not feel childish or overly cute.

The name should be interpreted as:
- warm but sharp
- playful but not light
- emotional but commercial
- minimal but memorable

### Brand Tone Keywords
- clean editorial
- soft contrast
- premium minimal
- sharp but warm
- quiet confidence

### One-Line Brand Definition
A white-based editorial portfolio site with sharp black contrast and a very restrained banana-toned accent.

---

## 4. Visual Direction

### Base Color Strategy
Use a white-based layout as the main foundation.

#### Recommended Palette
- Background: warm white or pure white
- Primary text: near black
- Secondary text: muted gray
- Accent: muted banana yellow
- Divider / UI line: very light gray

### Color Rules
- white is the main canvas
- black creates structure and contrast
- banana yellow is only a subtle signature accent
- do not use strong yellow as a primary brand color
- dark sections can appear only in selected areas, not as the full-site base

### Why White Base
The portfolio includes:
- dark product photography
- colorful food photography
- clean bright portraits
- airy space photography

A full dark theme would help Product but hurt Food, Portrait, and Space.

A white base gives the full portfolio a more flexible and premium frame.

---

## 5. Logo Direction

### Decision
Use the direction of the black-background logo as the base concept, but redesign it for a white-background website.

### Logo System
Create three versions:
1. Main logo
   - banana symbol + wordmark

2. Header logo
   - simplified wordmark
   - minimal, monochrome

3. Icon version
   - banana symbol only
   - used for favicon / thumbnail / small brand marks

### Logo Rules
- remove or avoid the current slogan for now
- keep the logo clean and more refined
- reduce rough texture if needed
- prioritize usability on a white background

## 8. Page Strategy

## Home
### Role
Brand-first landing page

### Purpose
- create a strong first impression
- present Banana Black as a premium creative portfolio
- guide users into Works or Contact
- show the four categories in a more curated way

### Recommended Sections
1. Fullscreen hero with strongest representative image
2. Short brand message
3. Category preview for Product / Food / Portrait / Space
4. Short studio positioning section
5. Inquiry CTA
6. Optional bottom CTA or selected works preview

### Home Rules
- hero must use image, not empty dark background
- avoid placing full contact info in the center of hero
- use a clear CTA such as:
  - View Works
  - Contact
  - Private Portfolio

---

## Works
### Role
Category hub page

### Purpose
- provide a clean overview of all four categories
- work as a curated entry point instead of a plain link list

### Recommended Structure
- page intro
- four category blocks
- each block includes one strong visual and short description
- optional direct links to each category page

---

## Product
### Role
Premium showcase page

### Strength
This is the strongest current category and should feel the most premium.

### Direction
- emphasize dark contrast and bold brand visuals
- allow stronger layout rhythm
- support featured shots and grouped series

### Notes
- avoid flat repetition only
- allow hero-like first image or featured project treatment

---

## Food
### Role
Commercial food and styling showcase

### Strength
Color, plating, and styling are already strong.

### Direction
- keep the page bright and appetizing
- feel polished, not casual
- optional filtering can be added later if useful

### Notes
Possible future filter examples:
- All
- Korean
- Dessert
- Beverage
- Brand Food

Filtering is optional, not MVP-critical.

---

## Portrait
### Role
People / profile / branding portrait showcase

### Direction
- keep it clean, light, and professional
- avoid making it feel like a plain ID-photo page
- support profile, actor, model, and branding portrait positioning

### Notes
A short intro sentence should explain the kind of portrait work offered.

---

## Space
### Role
Interior / spatial portfolio page

### Direction
- preserve openness and breathing room
- support wide shots and detail shots
- avoid repetitive same-angle grids

### Notes
Layout should help space photography feel spacious.

---

## About
### Role
Studio introduction page

### Purpose
- explain Banana Black’s perspective
- build trust
- introduce creative positioning without too much text

### Should Include
- short studio intro
- what kinds of work are focused on
- visual philosophy
- optional simple brand statement

---

## Contact
### Role
Conversion page

### Purpose
Make inquiry easy and intentional.

### Recommended Content
- short invitation to inquire
- inquiry form
- simple contact details
- optional fields such as:
  - brand / client name
  - category needed
  - desired timeline
  - reference availability
  - hidden portfolio request

### Important
This should not be a dead-end contact sheet.  
It should feel like a guided inquiry page.

---

## Hidden Portfolio
### Role
Private portfolio delivery page

### Purpose
Handle portrait-rights issues or client-only work sharing.

### Requirements
- not visible in main navigation
- shareable by direct link
- optional password protection later
- can be structured by project or category

When building this site, prioritize the following in order:

1. Strong homepage hero using an existing best photo
2. Premium white-based layout with black contrast
3. Clean typography and spacing
4. Better category structure than the current site
5. Simple but visible CTA flow
6. Mobile-friendly image layout
7. Elegant and restrained interactions

---

## 18. Important Output Rules

The output should not feel like a generic gallery template.

The website should:
- feel premium
- feel intentional
- feel minimal but warm
- make the existing photos look more valuable

Avoid:
- overdesigned layouts
- too many decorations
- strong yellow overuse
- dark mode across the whole site
- repetitive same-grid treatment for every category

This project is not about getting new photos first.  
It is about building a better frame for already strong work.

The redesign should make Banana Black feel more premium, more intentional, and more memorable while keeping the structure simple.

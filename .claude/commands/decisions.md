이 프로젝트의 영속 결정·findings를 참조하는 명령어다.

수행:
1. `docs/DECISIONS.md`를 읽는다(없으면 "아직 DECISIONS 없음"이라고 알리고 생성 제안).
2. 인자로 주제를 줬으면($ARGUMENTS) 관련 결정 항목만 추려 요약한다. 없으면 최신순 제목만 짧게 나열한다.
3. 로컬 메모리에 DECISIONS로 미러링 안 된 새 항목이 있으면 짚고, 미러링할지 묻는다.

원칙: 로컬 메모리는 기기 전용이라 포맷 시 소실되므로 중요한 결정은 git DECISIONS에 보관한다. 새 메모리는 DECISIONS에 날짜 항목(결정/이유/영향, 3줄)으로 미러링한다. 토큰 아껴 짧게.

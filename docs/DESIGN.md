# Design

## Source of Truth

상세 브랜드와 UX 원본은 `../PRD.md`다.

이 문서는 AI 작업자가 디자인 맥락을 빠르게 확인하기 위한 지도다.

## Brand Direction

Banana Black은 귀엽거나 가벼운 느낌보다, 따뜻하지만 날카롭고 상업적이면서 감정적인 프리미엄 포트폴리오 톤을 지향한다.

핵심 키워드:

- clean editorial
- soft contrast
- premium minimal
- sharp but warm
- quiet confidence

## Visual System

- 흰색 계열을 주 캔버스로 쓴다.
- 검정은 구조와 대비를 만든다.
- 바나나 옐로우는 강한 주색이 아니라 절제된 시그니처 액센트다.
- 전체 다크 테마는 피하고, 필요한 구간에서만 어두운 대비를 허용한다.
- 사진이 경험을 이끌고 UI는 사진을 방해하지 않는다.

## Page Principles

- Home은 강한 첫인상과 브랜드 포지셔닝을 담당한다.
- Works는 단순 목록보다 큐레이션된 진입점이어야 한다.
- Product는 가장 프리미엄하고 강한 대비를 허용한다.
- Food, Dessert, Portrait, Space는 각 카테고리의 성격을 살리되 하나의 브랜드 안에 있어야 한다.
- Contact는 문의 전환을 명확하고 자연스럽게 만든다.

## Mobile Rules

- 360, 390, 430 폭을 우선 검수한다.
- 주요 카피는 의도한 줄바꿈을 유지한다.
- 모바일에서 CTA와 첫 이미지 사이 간격을 깨지 않는다.
- fixed header와 works-local-nav가 겹치지 않아야 한다.
- 푸터 연락처와 사업자 메타 문구는 모바일 2줄 기준을 유지한다.

## Non-goals

- 과한 노란색 중심 브랜딩
- 사진보다 UI가 먼저 보이는 장식
- 일반 웹빌더 같은 포트폴리오 느낌
- 내부 운영 페이지를 공개 경험처럼 꾸미거나 노출하는 것

## Update Rules

브랜드 톤, 컬러, 페이지 역할, 모바일 줄바꿈 기준이 바뀌면 `PRD.md`, `docs/DESIGN.md`, 관련 HTML/CSS를 함께 확인한다.
